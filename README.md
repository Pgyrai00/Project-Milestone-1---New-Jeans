## NEWJEANS Fan Page

This is a simple, minimalistic fan site for **NEWJEANS**, built with HTML and CSS.
It includes:

- **HOME PAGE** (`index.html`) with an overview of the group, their group photo and an embedded performance vide
- **GALLERY PAGE** (`gallery.html`) showcasing the members of the group.
- **ALBUMS PAGE** (`albums.html`) showcasing discography information.
- **ASSETS FOLDER** (`assets`) containing CSS componets and images used.
